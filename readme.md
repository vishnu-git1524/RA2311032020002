# Backend Project

## Overview

Project contains a simple backend setup with a modular structure.
The focus is on clean organization, basic service logic, and consistent logging across components.

## Setup

```bash
npm install
node app.js
```

## Notes

* Built using Node.js and Express
* Organized into separate modules for clarity
* Logging middleware is used to track key operations
* Designed to be easy to run and test locally
* No external algorithm libraries are used
